## 🚀 Cómo Ejecutar

### Instalación Local (3 pasos)

```bash
# 1. Abrir terminal en la carpeta del proyecto
cd C:\Users\senis\OneDrive\Documentos\webs\Spa-Termas-Dayman

# 2. Instalar dependencias
npm install

# 3. Iniciar servidor de desarrollo
npm run dev
```

**Resultado**: Abre automáticamente en `http://localhost:5173`

### Build para Producción

```bash
npm run build
```

**Resultado**: Genera carpeta `dist/` lista para desplegar
